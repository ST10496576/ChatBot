# Cybersecurity Awareness Chatbot (Part 1)

## Overview
This project is a console-based chatbot built in C#. It is designed to educate users about basic cybersecurity concepts in an interactive way.

The chatbot provides information on topics such as phishing, password safety, and safe browsing practices.

---

## Features

- Personalised user greeting
- Voice greeting using a WAV audio file
- ASCII art interface for improved user experience
- Interactive menu system with cybersecurity topics
- Input handling for both numbers and keywords
- Typing effect for chatbot responses
- Input validation with fallback responses for incorrect input
- Error handling for missing audio files

---

## Topics Covered

- Phishing awareness
- Password safety
- Safe browsing and links
- General cybersecurity awareness
- Help and guidance commands

---

## How It Works

1. The user enters their name
2. A voice greeting is played
3. A menu is displayed with options
4. The user selects a topic using numbers or keywords
5. The chatbot responds with relevant information

---

## Technologies Used

- C#
- .NET Console Application
- System.Media (for audio playback)
- Console UI formatting
